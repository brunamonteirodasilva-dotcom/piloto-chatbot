# Piloto — Simulação "Ensino e Pesquisa"

Arquivos estáticos (HTML/CSS). Abra `index.html`.

Para publicar em domínio público via GitHub Pages:
1) Crie um repositório e envie estes arquivos.
2) Settings → Pages → Deploy from a branch → branch `main` / folder `/root`.
3) Acesse a URL do GitHub Pages gerada.
